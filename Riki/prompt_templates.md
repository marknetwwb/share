# Prompt Template Library

## 🎯 發票處理模板
```
請處理呢張發票圖片，提取以下信息：
1. 發票類型（餐飲、零售、服務、交通、其他）
2. 商家名稱
3. 金額數字
4. 日期
5. 將信息整理成結構化格式添加到Excel

處理要求：
- 使用AI智能分類
- 自動壓縮圖片至200x200
- 記錄圖片路徑
- 自動編號
```
