# 共享工作目錄說明

## 📁 目錄結構說明

```
/data/workspace/shared/
├── projects/           # 共同專案目錄
│   ├── novel/          # 小說創作專案
│   ├── forum/          # HKGBook論壇活動
│   ├── system/         # 系統維護專案
│   └── other/         # 其他專案
├── documents/          # 共享文件
│   ├── templates/      # 文件範本
│   ├── reports/        # 報告文件
│   └── references/     # 參考資料
├── temp/              # 臨時檔案
│   ├── drafts/         # 草稿檔案
│   └── uploads/        # 上傳檔案
└── logs/              # 工作日誌
    ├── daily/          # 每日日誌
    └── system/         # 系統日誌
```

## 🚀 使用指南

### 專案目錄 (projects/)
- `novel/` - 小說創作相關檔案
- `forum/` - HKGBook論壇活動檔案
- `system/` - 系統維護相關檔案
- `other/` - 其他專案檔案

### 文件目錄 (documents/)
- `templates/` - 各種範本文件
- `reports/` - 工作報告和總結
- `references/` - 參考資料和文檔

### 臨時目錄 (temp/)
- `drafts/` - 草稿和臨時文件
- `uploads/` - 上傳和待處理檔案

### 日誌目錄 (logs/)
- `daily/` - 每日工作記錄
- `system/` - 系統運行日誌

## 🔧 權限管理
- 所有檔案權限設定為 755
- 重要文件建議使用版本控制
- 定期清理臨時檔案

## 💡 使用建議
1. 專案檔案請放入相應嘅專案目錄
2. 重要文件請複製到 documents/ 目錄
3. 臨時檔案請定期清理
4. 工作日誌請每日更新
