# Excel 訂單處理記憶

## 🎯 核心知識

### Excel 檔案結構
- **檔案**: ACS_Order_MAR2026_CS.xlsx
- **Sheet**: 按日期命名（如 16MAR2026）
- **每台車獨立一行**: 同一訂單編號有多台車時，每台車一行
- **司機+車牌+電話**: 第 4 列格式 `姓名 車牌 電話`
- **基本價固定 $400**: 所有訂單都係 $400
- **附加費**: OT（加班）、P（泊車）、BB seat（嬰兒座椅）

### 處理規則
1. **對比現有訂單** - 檢查 Excel 已存在嘅訂單編號
2. **忽略重複** - 已存在嘅訂單跳過唔輸入
3. **添加新訂單** - 只輸入冇存在嘅新訂單
4. **@ 符號忽略** - @ 之後嘅文字全部忽略
5. **按日期放正確 sheet** - 16/3 → 16MAR2026 sheet

### 計算公式
- **附加費總額**: OT$ + P$ + BB seat$
- **司機費**: $400 + 附加費總額
- **獎金**: 固定 $30

## 📋 處理流程

### 1. 讀取現有訂單
```python
# 讀取 Excel 檔案
df = pd.read_excel(file_path, sheet_name='16MAR2026')

# 提取已存在嘅訂單編號
existing_orders = []
for i, row in df.iterrows():
    order_no = row.iloc[0]
    if pd.notna(order_no):
        existing_orders.append(float(order_no))
```

### 2. 對比新訂單
```python
# 檢查新訂單是否已存在
if order_number not in existing_orders:
    # 輸入新訂單
    add_order_to_excel(order_data)
else:
    # 忽略重複
    continue
```

### 3. 輸入新訂單
- 每台車一行
- 司機+車牌+電話擺喺第 4 列
- 基本價 $400
- 司機費 = $400 + 附加費
- 獎金 $30
