# Excel 訂單處理指令

## 每月 Excel 訂單處理任務

### 📝 任務描述
處理 ACS_Order_MAR2026_CS.xlsx 檔案，根據提供嘅訂單資料：
- 對比 Excel 已存在嘅訂單編號
- 忽略重複訂單
- 只輸入冇存在嘅新訂單
- 按正確格式輸入到對應嘅日期 sheet

### 🔧 處理規則
1. **每台車獨立一行** - 同一訂單編號有多台車時，每台車一行
2. **司機+車牌+電話** - 擺喺第 4 列，格式：`姓名 車牌 電話`
3. **基本價固定 $400** - 所有用戶基本價都係 $400
4. **附加費計算** - OT（加班）、P（泊車）、BB seat（嬰兒座椅）
5. **@ 符號忽略** - @ 之後嘅文字全部忽略
6. **按日期放正確 sheet** - 16/3 → 16MAR2026 sheet

### 📋 Excel 欄位格式
| 列數 | 內容 | 格式 |
|------|------|------|
| 1 | 訂單編號 | 數字 |
| 2 | 時間 | HH:MM:SS |
| 3 | 路線 | XXX > XXX |
| 4 | 司機+車牌+電話 | 姓名 車牌 電話 |
| 5 | 狀態 | Done |
| 6 | 開始時間 | HH:MM:SS |
| 7 | 結束時間 | HH:MM:SS |
| 10 | 附加費 | OT$X, P$Y, BB seat$Z |
| 11 | 附加費金額 | 數字 |
| 12 | 基本價 | 400.0 |
| 13 | 司機費 | $400 + 附加費 |
| 14 | 獎金 | 30.0 |

### 💡 處理步驟
1. 讀取指定日期嘅 Excel sheet
2. 提取所有已存在嘅訂單編號
3. 對比新訂單資料
4. 跳過重複訂單
5. 按格式輸入新訂單
6. 保存更新後嘅檔案

### 🎯 輸出示例
```
訂單編號：546120
時間：02:30:00
路線：MB300 > TST
司機+車牌+電話：Eva TZ9800 97062468
狀態：Done
開始時間：02:53:00
結束時間：03:21:00
附加費：OT$60, P$35, BB seat$80
附加費金額：175.0
基本價：400.0
司機費：575.0
獎金：30.0
```

### ⚠️ 注意事項
- 保持現有訂單完全唔變
- 只添加新訂單，唔修改舊有資料
- 每台車都要獨立一行
- 按 Excel 原有格式輸入
- 忽略所有 @ 符號後嘅文字
