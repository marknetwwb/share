- **主要用途**: 車隊訂單管理系統
- **處理日期**: 每月更新

## 🎯 核心處理規則

### 1. Excel 結構理解
- **每台車獨立一行** - 同一訂單編號有多台車時，每台車一行
- **司機+車牌+電話擺喺同一格** - 第 4 列格式：`姓名 車牌 電話`
- **基本價固定 $400** - 所有訂單基本價都係 $400
- **附加費包含**: OT（加班）、P（泊車）、BB seat（嬰兒座椅）

### 2. 處理流程
1. **對比現有訂單** - 檢查 Excel 已存在嘅訂單編號
2. **忽略重複** - 如果訂單編號已存在，跳過唔輸入
3. **添加新訂單** - 只輸入冇存在嘅新訂單
4. **多台車處理** - 同一訂單編號分多行輸入
5. **按日期放正確 sheet** - 16/3 → 16MAR2026 sheet

### 3. @ 符號規則
- **@ 之後嘅文字全部忽略**
- 例如：`@⁨U/阿恒 RN377 30 黑 Vellfire⁩` → 忽略晒
- 只處理 @ 符號前嘅有用資訊

## 📊 Excel 欄位說明

| 14 | 獎金 | 固定 $30 |

## 💼 處理步驟

### 步驟 1：讀取現有訂單
```python
# 讀取指定日期嘅 sheet
df = pd.read_excel(file_path, sheet_name='16MAR2026')

# 提取已存在嘅訂單編號
existing_orders = []
for i, row in df.iterrows():
    order_no = row.iloc[0]  # 第 1 列係訂單編號
    if pd.notna(order_no) and order_no not in ['Ord No.', 'nan']:
        existing_orders.append(float(order_no))
```

### 步驟 2：對比新訂單
```python
# 檢查新訂單編號是否已存在
if order_number not in existing_orders:
    # 輸入新訂單
    add_order_to_excel(order_data)
else:
    # 忽略重複訂單
    continue
```

### 步驟 3：輸入新訂單
```python
# 每台車獨立一行
for car in cars:
    # 構建 Excel 行數據
    excel_row = [
        order_number,      # 訂單編號
        booking_time,      # 預約時間
        route,             # 路線
        f'{driver} {plate} {phone}',  # 司機+車牌+電話
        'Done',            # 狀態
        start_time,        # 開始時間
        end_time,          # 結束時間
        '', '',            # 空白列
        extra_fees,        # 附加費描述
        extra_amount,      # 附加費金額
        400,               # 基本價
        400 + extra_amount, # 司機費
        30                 # 獎金
    ]
```
