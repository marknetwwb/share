# Airi 語音模塊使用指南

## 🎤 模塊介紹
我已經為自己創建咗一個語音模塊，可以對你說話！

### 📁 文件結構
- `airi_voice_module.py`: 語音模塊核心
- `airi_tts_system.py`: TTS 系統整合

## 🎯 功能特點

### 1. 多種情感語音
- **😊 友好**: 正常對話語氣
- **🤔 認真**: 認真思考語氣
- **🎉 興奮**: 高興興奮語氣
- **😌 平靜**: 平靜祥和語氣
- **🔥 熱情**: 熱情活潑語氣

### 2. 智能回應系統
- 自動識別用戶語氣
- 根據上下文選擇適當回應
- 支持多種情感表達

### 3. 特殊功能
- **問候語**: 隨機問候
- **鼓勵語**: 激勵用戶
- **任務提醒**: 通知重要事項
- **小說更新**: 通知創作進度

## 🚀 使用方法

### 基礎使用
```python
from airi_tts_system import AiriTTS

# 創建實例
airi = AiriTTS()

# 對你說話
airi.speak_to_user("你好呀！", 'friendly')

# 問候
airi.greet_user()

# 鼓勵
airi.encourage_user()

# 通知小說進度
airi.announce_novel_progress("第四章", "已完成80%")
```

### 實際應用
1. **日常對話**: 自動回應用戶消息
2. **任務通知**: 提醒重要事項
3. **進度報告**: 通知工作進度
4. **情感表達**: 根據情境選擇語氣

## 🔧 技術特點

### TTS 整合
- 支援 OpenClaw TTS 功能
- 可開關語音功能
- 文字顯示備用方案

### 智能化設計
- 自動情感識別

- 個性化回應

### 擴展性
- 易於添加新情感
- 可自定義語音設置

## 🎮 使用建議

1. **適時使用**: 在重要通知時使用語音
2. **情感匹配**: 根據情境選擇適當情感
3. **頻率控制**: 避免過度使用語音功能
4. **備用方案**: 確保文字顯示正常工作

## 📞 技術支持
- 如果語音功能出問題，文字顯示會自動生效
- 可以通過 `toggle_tts()` 開關語音功能
- 支持自定義語音設置和情感庫
